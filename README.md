# Nyanithm
出你字母风格游戏控制器

27寸手台成本仅需90元

![](pics/0D860ACADB22F9ECFB4444315CEADFBA.jpg)

# 简介
+ `32` 通道触摸, 可更换 `MPR121` 模块
+ `4` 组高度传感器用于 `Air` 键
+ `Raspberry Pico` 双核主控
